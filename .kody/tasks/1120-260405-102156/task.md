# LaTeX Rendering Failure in Chat

Description

Title: Chat interface fails to render LaTeX equations; displays raw syntax markers.

User Role: Student / End-User

Precondition: User is logged into the chat interface and has an active conversation.

Steps to Reproduce:

Open any chat window.

Enter a message containing standard LaTeX syntax (e.g., $E=mc^2$ or $$\frac{-b \pm \sqrt{b^2-4ac}}{2a}$$).

Send the message.

Observe the output in the chat bubble.

Expected Result: The system should parse the LaTeX markers and render a formatted mathematical equation.

Actual Result: The chat displays the raw text including the math signs (e.g., showing the literal characters instead of the visual formula).
