Mathematical expression in lesson chat is displayed as plain text instead of formatted math
# 🐞 Bug Report

## 1. Title
Mathematical expression in lesson chat is displayed as plain text instead of formatted math

## 2. Environment
- Environment: prod
- Browser / Device: Chrome
- User Role / Tenant: Student

## 3. Preconditions
User is logged into the platform.

A lesson contains a chat interaction with a mathematical expression.

The expression is written in a math format (LaTeX / math syntax).

## 4. Steps to Reproduce
Open a lesson that contains a chat interaction.

Navigate to the chat section with a mathematical expression.

Observe how the formula is displayed.

## 5. Expected Result
Mathematical expressions should be rendered in proper mathematical formatting (e.g., using MathJax / KaTeX).

The formula should appear visually formatted rather than as raw text.

## 6. Actual Result
The mathematical expression appears as plain text.

Mathematical notation is not rendered visually.

## 7. Reproducibility
always
